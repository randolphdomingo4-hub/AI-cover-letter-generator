# AI Cover Letter Generator V1

A beginner AI-powered Python application that generates professional cover letters based on user input using Google's Gemini API.

## Features

- Accepts applicant name
- Accepts target role
- Accepts skills
- Accepts experience
- Generates AI-powered cover letters
- Uses Google Gemini API
- Streamlit web interface

## Skills Used

- Python fundamentals
- Functions
- User input handling
- API integration
- Prompt engineering
- Google Gemini API
- Streamlit
- AI application development
